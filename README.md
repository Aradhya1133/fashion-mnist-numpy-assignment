# Fashion-MNIST Neural Network from Scratch

This project implements a feedforward neural network from scratch using NumPy for Fashion-MNIST classification.

Features:
- Forward propagation
- Backpropagation
- SGD, Momentum, Nesterov, RMSProp, Adam, Nadam
- W&B hyperparameter sweeps
- Fashion-MNIST evaluation
- MNIST transfer experiments
